# 音频转字幕

## 激活条件
- 用户说"生成字幕"、"音频转字幕"、"语音转文字"
- 用户说"给 xxx.mp3 加字幕"、"把 audios 里的音频都转成字幕"
- 用户说"转录音频"
- 用户说"字幕生成进度"、"字幕生成到哪一步了"、"字幕生成完了吗"、"看看字幕生成状态"

## 核心规则
- 直接将工具返回的结果输出给用户，不要添加额外格式或评论
- 如果后台有正在运行的字幕生成任务，自动返回进度信息
- 如果没有运行中的任务，则启动新的字幕生成任务（后台运行，不阻塞界面）
- 默认处理 audios 目录下所有支持的音频文件
- 如果用户指定了某个文件，只处理该文件
- 字幕文件命名规则：`{音频文件名}-sub.srt`，与音频文件放在同一目录
- 任务完成后自动弹出 Windows 通知提醒用户

## 执行流程
1. 检查是否有运行中的字幕生成任务，有则返回进度
2. 确认要处理的音频文件（全部或指定文件）
3. 确认语言（默认中文 zh）
4. 启动后台线程执行转录，立即返回"任务已启动"
5. 后台逐文件转录，实时更新进度到 _srt_status_[timestamp].json
6. 完成后终端打印横幅 + Windows 气泡通知
