# 企业微信消息发送

## 激活条件
- 用户说"发微信给xxx"、"给xxx发一条企业微信"
- 用户说"通知xxx"、"提醒xxx"（配合企业微信）
- 用户说"发送消息给xxx"

## 核心规则
- 直接将工具返回的结果输出给用户，不要添加额外格式或评论
- 如果用户没有指定接收人（touser），询问用户要发给谁

## 执行流程
1. 确认接收人 UserID 和消息内容
2. 调用 wecom_message skill
3. 将结果直接输出给用户
