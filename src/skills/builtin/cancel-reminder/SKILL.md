# 取消定时任务

## 激活条件
- 用户说"取消提醒"、"取消定时任务"、"取消刚才的提醒"
- 用户说"取消任务1 2 4"（多选取消）
- 用户说"取消所有任务"、"取消全部定时"、"全部取消"
- 用户说"不要提醒了"、"把提醒删掉"

## 核心规则
- 若用户说"取消所有"/"取消全部"，设置 cancel_all=true
- 若用户说"取消任务1 2 4"，提取序号设置 task_indices=[1, 2, 4]
- 若用户提供任务ID，使用 task_id 精确定位
- 取消成功后从 MySQL 和内存中同时移除
- 直接将工具返回的结果输出给用户，不要添加额外格式或评论

## 执行流程
1. 判断用户意图：取消全部 / 多选取消 / 单个取消
2. 调用 cancel_reminder skill 执行对应操作
3. 将结果直接输出给用户

## 输出模板
```
# 取消全部
✅ 已取消全部 3 个定时任务。

# 多选取消
✅ #1 提醒喝水
✅ #2 发送邮件
⚠️ 序号 #4 超出范围

# 单个取消
✅ 已取消定时任务（ID: abc12345）
```
