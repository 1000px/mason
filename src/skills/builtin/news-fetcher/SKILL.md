# 新闻抓取

## 激活条件
- 用户询问"今天有什么新闻"、"最新科技新闻"、"热点新闻"
- 用户提到"V2EX"、"新闻摘要"、"科技热点"
- 用户想了解当前技术社区的热门话题

## 核心规则
- 默认抓取 5 条新闻，用户可指定数量
- 每条新闻包含标题、链接和内容预览
- 抓取失败时告知用户原因
- 不编造新闻内容，所有数据来自 V2EX API

## 执行流程
1. 确认用户需要的新闻条数（默认 5 条）
2. 调用 news_fetcher skill 从 V2EX API 抓取热门话题
3. 整理为结构化摘要格式
4. 返回给用户

## 输出模板
```
📰 最新 V2EX 热点：

标题: {title}
链接: {url}
预览: {preview}
---
```
