# 列出定时任务

## 激活条件
- 用户说"列出定时任务"、"查看提醒列表"、"有哪些定时"
- 用户说"当前任务"、"待执行的任务"、"我的提醒"
- 用户想了解已设置了哪些定时任务

## 核心规则
- 调用 list_reminders 工具获取结果
- **直接将工具返回的结果原样输出给用户，不要添加任何额外格式或评论**
- 不要用 markdown 重新排版，不要添加"当前系统中有一个定时任务"之类的总结语

## 执行流程
1. 调用 list_reminders skill 查询数据库
2. 将工具返回的文本直接输出，不做任何修改

## 输出模板
```
📋 当前定时任务列表：
1. [ID: abc12345] 2026-05-13 21:00 - 提醒用户喝水
2. [ID: def67890] 2026-05-13 22:30 - 发送邮件给老板
```
