# 定时任务提醒

## 激活条件
- 用户说"1分钟后提醒我"、"5分钟后叫我开会"、"定时发送消息"
- 用户提到"提醒"、"定时"、"闹钟"、"到时间"等关键词
- 用户需要延迟执行某个操作

## 核心规则
- 必须提供时间描述（如"1分钟后"、"下午3点"）
- 必须提供任务描述（要执行什么操作）
- 时间已过则提示用户重新设置
- 时间解析失败时告知用户并提供正确格式示例

## 执行流程
1. 解析用户提供的时间描述
2. 确认任务内容和参数
3. 设置定时器，到时间后自动触发任务
4. 向用户确认定时任务已设置成功

## 输出模板
```
✅ 已设置定时任务：{time_description} 后将执行：{task_description}
```
