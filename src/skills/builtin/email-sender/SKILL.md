# 邮件发送

## 激活条件
- 用户说"发邮件给xxx"、"帮我发送一封邮件"
- 用户提到"邮件通知"、"email"、"发送到邮箱"
- 用户需要将内容通过邮件发送给某人

## 核心规则
- 必须提供收件人邮箱地址
- 必须提供邮件正文内容
- 邮件主题可选，默认为"Mason 提醒"
- 发件人邮箱和授权码需提前在环境变量中配置
- 发送失败时告知用户具体错误原因

## 执行流程
1. 确认收件人邮箱地址
2. 确认邮件主题和正文内容
3. 通过 SMTP 服务器发送邮件
4. 返回发送结果

## 输出模板
```
✅ 邮件已成功发送至 {receiver}
```
